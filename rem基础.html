em是等于父元素字体大小的单位，rem是相当于html元素的字体大小 media query 是css3，根据屏幕尺寸设置不同样式， mediatype：all/scree/print and/not/only 三个关键字 媒体特性：width、max-width、min-width 根据不同屏幕尺寸引入css ： @media screen and (max-width:800px) { body { background-color: pink; } } 意思就是在屏幕上，并且最大的宽度是800px @media screen and (max-width:800px) and (min-width:500px) 但其实可以利用层叠性，如果后面还有一段min-width=801或许可以只写min 一个括号写一个条件，括号包含等号
